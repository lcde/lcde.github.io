01_AutoCAD绘图技巧分析

Published 2016-08-13

Contents
  1. 1. 本小节知识点：
    1. 1.1. 1.直线、圆的键盘运用技巧
    2. 1.2. 2.修改平面图的键盘运用技巧

本文对AutoCAD 2008 版本有一定绘图技巧的人，通过本文，可以使读者在绘图速度上有一定的提高。
本文主要加深画图者运用键盘的能力。前期要慢慢培养成集合键盘的辅助。发出命令，记住命令的缩写
单词，是必不可少的。比如直线的英文是Line，所以AutoCAD的直线命令就是键盘上L（大小写无所谓）。
当然了，有些命令会多余两个字母，需要大家多下点功夫，多记忆，多操作。运用熟练后，会大大加快
大家的绘图速度。全文以减速箱零件图的绘制为主要任务。

本小节知识点：

  1. 【掌握】直线、圆的键盘运用技巧

  2. 【掌握】修改平面图的键盘运用技巧

  3. 【了解】画图环境的个性设置

  4. 【掌握】平面图转三维图的键盘运用技巧

1.直线、圆的键盘运用技巧

AutoCAD在绘制时，要对软件发出命令时，主要有两个方式

  • 通过鼠标选择命令条上的命令按钮
  • 通过键盘输入英文字母，即软件约定好的命令缩写字母
  • 直线的绘制：

在这里主要讲第二种方式，就是通过键盘输入英文字母。接下来画一段100mm的线段为例。

在软件没有命令要执行的情况下，即在空闲状态。

输入英文字母l（要在输入法是英文状态下输入，若要使打出汉字，软件识别不出命令，更不会去执行）。
一般在下面的命令栏里面会出现一个英文字母l。之后点击Enter或空格键，这时会看到命令栏里显示出软
件系统提示的命令。

这时，有两种方式去画一条线段。

  • 第一种：鼠标点击到一点，命令栏里面会提示画图者去“指定下一点或者放弃（U）”。当第二点用鼠标点指定后，线段的长度
    即确定。当然了，接下来不用输入命令，就可以指出下一点的位置，画出无数的首尾相连的线段。

  • 第二种：鼠标点击到下一点，接下来输入100，然后鼠标给线段一个方向，不需要点击鼠标，在方向确定下来后（会有一条虚线
    出来），点击键盘上的Enter或空格键表示确定。

总结：

  • 软件在不确定是否为工作状态的时候，点击Esc键，可以退出工作状态
  • Windows下多数快捷键，在AutoCAD中配合命令来使用可以提供绘图速度。你还可以根据自己的工作环境定制一些快捷键
  • 在鼠标配合键盘制图的过程中，时刻要看命令栏里输入的命令以及软件提示绘图者接下来要做哪些工作
  • 了解命令的参数，例如：绘制线段过程中，需要知道线段的起点坐标，重点坐标或者说是直线的起点，方向，长度等等
  • 数学知识：线段的性质：两端都是有端点，不可延长，有别于直线，射线
快捷键 作用
Ctrl + C 复制
Ctrl + V 粘贴
Ctrl + X 剪切
Ctrl + Z 撤回

2.修改平面图的键盘运用技巧

我们已经学会如何快速的绘制直线和圆，但是在我们的图纸中，需要不断的修改。在这里介绍一些快捷修改方式

快捷键 作用
Ctrl + Z 撤回
Delete 删除
Ctrl + Z 剪刀命令
MA 格式刷

注意点： 简单的平面图可以选择MA命令来操作，但是大型复杂的图形，存在很多样条，颜色的零件图，不建议使用。

下面我就用具体案例讲解修改平面图的技巧：

  • 首先确定，我们结合三个视图画出左上角的视图。我们先驱动上面的两个同心圆的位置，画个十字交叉的点划线。之后画出
    两条相互垂直的实线，然后以两条的交点画两个同心圆，者两个圆的半径有视图可以找到。分别是半径是15，30.画出即可：

  • 然后再找下底面的线，这里面有个操作技巧，也会经常用到的。在找最下面的线时，我们完全可以画出一条新的线。当然也可以使用已有的线段通过平行移动到他的位置。这样我们节省了一步画线段的步骤。比如：选中同心圆的水平线，会在这条线的两段和中点出现三个蓝色的夹点，我们可以用鼠标点中任意的一个点，会变成红色，比如下图。然后命令栏里面会提示“指定拉伸点或【几点（B）/复制（C）/放弃（U）/推出（X）】”：说明我们可以选择四个命令，基点的意思是，重新选择这条直线的红色的夹点。复制是将这条直线复制到另外一个地点，原直线不变。新的直线只需要给定一点既可以确认，因为既然是复制，就表明了和原来的直线是平行的。当然了，我们可以用鼠标选定一个方向，输入距离既可以了。要想不然原直线消失，就必须在输入距离之前，输入键盘上的字母“ C”，表明是复制原直线。

  • 在出现拉伸的命令时，我们可以一直按空格键，可以依次跳出拉伸、移动、旋转、比例缩放、镜像，循环出现。这个命令是不是很强大。
